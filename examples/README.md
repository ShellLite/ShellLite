# ShellLite Examples

* **[ShellLite-Website](github.com/ShellLite/ShellLite-Website):** The official project website, complete with an interactive online compiler, built mostly ShellLite's native web server capabilities.
* **[ChessEngine](github.comShellLite/ShellLiteChessEngine):** A fully functional, Chess Engine and lichess bot written purely in ShellLite.
